## 📌 What is Prototyping?
Prototyping is the process of building an **early interactive model** of a 
product to test and validate ideas before full-scale development. It is a 
critical step in **Design Thinking** that saves time, cost, and effort.

## 🔄 Design Thinking Process Followed
1. **Empathise** — Understood student pain points around study management
2. **Define** — Defined the problem: students lack an organised study tool
3. **Ideate** — Brainstormed UI solutions and app features
4. **Prototype** — Built interactive screens in Figma
5. **Test** — Reviewed screens for usability and flow
